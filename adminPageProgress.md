# 관리자 페이지 제작 진행도 (Admin Page Progress)

본 문서는 NESS 챗봇의 관리자 페이지 제작을 위한 일정, 진행도 및 세부 구성을 추적하는 문서입니다.

---

## 🎯 전체 목표

- **접근 제어**: URL 직접 접속 시 로그인 페이지로 이동, 인증된 관리자만 접속 가능하도록 보호
- **주요 기능**:
  1. **지점 등록**: 거주 지역 기반의 가까운 잡모아 센터 안내 기능 관리
  2. **관리자 계정**: 시스템 설정 권한을 가진 관리자 계정 생성 및 관리
  3. **Gemini API**: 챗봇의 뇌 역할을 하는 AI 모델 연동 및 키 관리
  4. **프롬프트 통합 관리**: 시나리오, 인트로/안내, PDF 규정 지식 등을 문법 오류 없는 폼 에디터로 제어

---

  - **거주지 단계 하드코딩**: 필수 단계인 '지역 선택'을 로직으로 분리하여 관리자가 실수로 삭제할 수 없도록 보호 완료

## 🛠 세부 페이지 구성 및 진행도 (Components & API)

### 1. 공통 로그인 및 인증

- **상태**: 🟩 완료 (100%)
- **기능**: JWT 기반 보안 인증, Middleware 라우팅 가드

### 2. 지점 등록 및 관리자 관리

- **상태**: 🟩 완료 (100%)
- **기능**: 지역별 지점 CRUD, 관리자 계정 활성화/비활성화 제어

### 3. Gemini API 및 시스템 설정

- **상태**: 🟩 완료 (100%)
- **기능**: API Key 암호화(마스킹) 처리 및 실시간 연동

### 4. 프롬프트 및 시나리오 통합 관리 (Core)

- **상태**: 🟩 완료 (100%)
- **고도화 내용**:
  - **GUI 에디터**: 질문 시나리오, 인트로, 안내 메시지 등 복잡한 JSON 데이터를 개발 지식 없이 폼 형태로 안전하게 수정 가능
  - **PDF 지식 통합**: 외부 파일에 의존하던 규정 내용을 DB화하여 관리자 화면에서 즉시 업데이트
  - **동적 템플릿**: 관리자가 추가한 커스텀 텍스트 항목을 시스템 프롬프트에 즉시 주입 가능
  - **시스템 안정성**: 데이터가 비어 있는 경우에도 시스템 기본값(Default)을 자동으로 로드하여 서비스 중단 방지
