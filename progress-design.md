## 🎯 랜딩 페이지 디자인 보완 상세 계획

### Session 1: 애니메이션 스크롤 이벤트 연동 (Scroll Reveal)

**목표**: 전체 섹션이 이동할 때 스크롤 이벤트를 추가하여 스크롤을 내리면 한 섹션씩 부드럽게 나타나도록 애니메이션 업데이트.

- [x] Framer Motion 또는 Intersection Observer를 활용한 스크롤 애니메이션 훅 적용
- [x] 각 섹션(Hero, Guide, Technology, Footer)에 스크롤 감지 및 등장 트랜지션 적용
- [x] 스크롤 시 콘텐츠가 위로 떠오르는(Reveal Up) 효과 일괄 적용

### Session 2: 히어로 섹션 (Hero Section) 고도화 및 가상 채팅 애니메이션

**목표**: 히어로 섹션을 2단 분할하여 좌측은 타이틀 및 CTA, 우측은 가상의 NESS 챗봇 채팅이 자동으로 진행되는 애니메이션 뷰를 추가합니다.

- [x] `page.tsx` 히어로 섹션 레이아웃 변경 (Grid flex-col -> 2 Column lg:grid-cols-2)
- [x] 우측 가상 챗봇 UI 컴포넌트 (`VirtualChatDemo.tsx` 등) 생성
- [x] 가상 챗봇 내 더미 메시지가 타이핑되며 스크롤되는 애니메이션 루프 구현
- [x] 히어로 섹션 전체 디자인 스케일 및 여백 최적화

### Session 3: 정보 요약 섹션 (Guide Section) 내용 및 디자인 개편

**목표**: 국민취업지원제도(NESS)가 어떤 제도인지 명확히 알 수 있도록 인포그래픽적 요소와 설명을 강화하여 디자인을 수정합니다.

- [x] NESS 제도의 핵심 목적(취업지원 및 생계안정)을 시각적으로 돋보이게 하는 UI 컴포넌트 적용
- [x] 카드형 UI(Type 1 구직촉진수당, Type 2 취업활동비용)의 레이아웃 확장 및 아이콘/일러스트 추가
- [x] 정보 전달력을 높이기 위한 타이포그래피 계층(Hierarchy) 재조정

### Session 4: 페이지 전체 폴리싱 및 반응형 점검

**목표**: 추가된 애니메이션과 개편된 섹션들이 데스크톱 및 모바일에서 어색함 없이 동작하도록 세밀하게 조정합니다.

- [x] 모바일 환경(소형 화면)에서 가상 채팅 애니메이션 크기 및 형태 최적화
- [x] 스크롤 애니메이션 딜레이 및 타이밍(easing) 조정으로 고급스러운 체감 속도 구현
- [x] 컬러 대비 및 다크 테마 글로우 효과 미세 조정
